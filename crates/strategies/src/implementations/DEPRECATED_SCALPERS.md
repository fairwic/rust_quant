# ⚠️ DEPRECATED STRATEGIES - Scalper Research 2026-06

本目录下的两个5分钟scalping策略经过详尽测试后确认**无法达到目标要求**，现已标记为废弃。

## 废弃策略清单

### 1. Range Reversion Scalper (`range_reversion_scalper/`)
- **逻辑**: Bollinger Bands + RSI 极值反转
- **状态**: 技术正确，但盈利不足
- **最佳成绩**: Win 64.2%, 月PnL +3u (目标20u)
- **详见**: `range_reversion_scalper/DEPRECATED.md`

### 2. Momentum Breakout Scalper (`momentum_breakout_scalper/`)
- **逻辑**: EMA趋势 + Pullback入场 + 3档trailing
- **状态**: 技术正确，但全配置亏损
- **最佳成绩**: Win 69.5%, 月PnL -4.15u
- **详见**: `momentum_breakout_scalper/DEPRECATED.md`

---

## 测试总结

### 数据规模
- **K线数量**: 34,560根 (BTC/ETH 5分钟)
- **时间跨度**: 2个月 (2026年5-6月)
- **配置测试**: 18,888+ 参数组合
- **周期测试**: 5分钟、15分钟
- **市场环境**: 强熊市 (BTC -23%, ETH -31%)

### 目标 vs 实际

| 指标 | 目标 | Range最佳 | Momentum最佳 | 达成度 |
|------|------|-----------|--------------|--------|
| 胜率 | >60% | 64.2% | 69.5% | ✅✅ |
| 频次 | ≥60/月 | 93.5/月 | 52.5/月 | ✅⚠️ |
| 回撤 | <10% | 3.4-10% | 8.1% | ✅✅ |
| **月PnL** | **≥20u** | **+3u** | **-4.15u** | **❌❌** |

### 核心问题

1. **费率拖累致命** (5分钟)
   - 典型波动: 0.3-0.5%
   - 往返费率: 0.1% (taker) 或 0.02% (maker)
   - 费率占比: 20-33% (taker) 或 4-7% (maker)
   - 即使零费率，Range策略毛利仅+4.85u/月

2. **市场环境不利**
   - 熊市趋势碾压均值回归的多头
   - Pullback入场在下跌中捕获死猫跳
   - 测试期内BTC/ETH持续下跌，策略设计假设被破坏

3. **数学天花板**
   ```
   目标: 20u/月 = 240%/年
   实际最佳: +3u/月 = 36%/年 (含2.7x杠杆，DD达极限)
   缺口: 6.7倍
   ```

4. **杠杆无法弥补**
   - 达到20u需18x杠杆
   - 但DD会爆炸到55%（目标<10%）
   - 约束冲突无解

## 尝试过的优化方向

✅ **参数扫描**: 15,000+ configs  
✅ **止盈止损比**: 对称、非对称、trailing  
✅ **方向偏向**: long-only, short-only, both  
✅ **费率敏感度**: maker(0.02%), taker(0.10%), 零费率  
✅ **杠杆测试**: 1x - 7x  
✅ **周期切换**: 5分钟、15分钟  
✅ **入场质量**: RSI极值、趋势过滤、波动过滤  
✅ **出场逻辑**: 单目标、三档trailing、移动止损

**结论**: 所有常规优化方向均已穷尽，无法弥补6.7倍PnL缺口。

## 替代方案建议

如需达到类似目标，建议：

### 短期可行
1. **更高周期**: 1小时、4小时、日线
   - 费率占比降低10-100倍
   - 趋势信号更可靠
   
2. **降低预期**: 月化5-10% (而非20%)
   - 符合市场中性策略的现实回报
   - 在可接受风险范围内

### 长期方向
1. **套利策略**: Funding rate arbitrage, cross-exchange
2. **多策略组合**: 降低单策略依赖与相关性
3. **外部alpha**: 订单簿L2、持仓量变化、巨鲸监控
4. **机器学习**: 特征工程 + 集成模型（但需海量数据）
