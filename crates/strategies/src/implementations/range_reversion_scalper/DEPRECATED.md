# ⚠️ DEPRECATED - Range Reversion Scalper Strategy

**状态**: 实验性 / 已废弃  
**原因**: 无法在目标约束下达到盈利要求  
**日期**: 2026-06

## 目标 vs 实际

| 指标 | 目标 | 最佳实现 | 状态 |
|------|------|----------|------|
| 胜率 | >60% | 64.2% | ✅ |
| 频次 | 高(≥60/月) | 93.5/月 | ✅ |
| 回撤 | <10% | 3.4-10% | ✅ |
| **月PnL** | **≥20u/100u** | **~3u/月** | **❌ 差距6.7×** |

## 测试覆盖

- **参数组合**: 15,000+ configs
- **数据量**: 34,560根K线 (BTC/ETH 5m, 2个月)
- **市场环境**: 强熊市 (BTC -23%, ETH -31%)
- **周期**: 5分钟、15分钟
- **费率**: maker(0.02%)、taker(0.10%)、零费率
- **杠杆**: 1x - 7x
- **方向**: long-only、short-only、both

## 最优配置

```rust
RangeReversionBacktestTuning {
    band_k: 2.5,              // Bollinger宽度
    rsi_long_max: 25.0,       // 超卖阈值
    rsi_short_min: 75.0,      // 超买阈值
    stop_atr_mult: 2.0,       // 止损2倍ATR
    target_atr_mult: 1.5,     // 止盈1.5倍ATR
    band_period: 20,
    rsi_period: 14,
    atr_period: 14,
    cooldown_candles: 2,
    allow_long: true,
    allow_short: true,
}
// + Maker fees (0.02%)
// + Leverage 2.7x (DD极限)
```

**结果**: Win 64.2%, Freq 93.5/月, DD ~10%, PnL ~3u/月

## 为什么失败

1. **费率拖累**: 5分钟典型波动0.3-0.5%，0.1%往返费率占比20-33%
2. **市场环境**: 强熊市中均值回归的多头被趋势碾压
3. **Edge上限**: 零费率下毛利仅+4.85u/月，任何真实费率都使其接近盈亏平衡
4. **杠杆约束**: 达到20u/月需18x杠杆，但DD会爆炸到55%

## 数学天花板

```
目标: 20u/月 = 20%/月 = 240%/年
实际edge (maker): +1.1u/月 = 1.1%/月 = 13.2%/年
Max viable leverage (DD<10%): 2.7x
Max achievable PnL: 1.1 × 2.7 = 3u/月

Gap to target: 20 / 3 = 6.7× 缺口
```

## 不推荐用于

- ❌ 生产环境实盘
- ❌ 5分钟周期
- ❌ 熊市环境
- ❌ 期望高频高盈利的场景

## 可能适用场景（未验证）

- ⚠️ 1小时及以上周期（费率占比降低）
- ⚠️ 牛市环境（多头顺势）
- ⚠️ 降低盈利预期至3-5%/月

## 替代方案

如需达到20%/月目标，需考虑：
1. 更高时间周期策略（日线、4小时）
2. 套利策略（funding rate arbitrage）
3. 多策略组合（降低相关性）
4. 引入外部alpha（订单簿、持仓量、资金费率）

---

**结论**: 策略技术实现完整且正确，但在给定约束下无法达到盈利目标。标记为实验性，不推荐部署。
