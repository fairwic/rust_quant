## 市场阶段策略

### 熊市阶段

| 阶段 | 特征 | 策略偏向 | 做多杠杆 | 做空杠杆 | 仓位上限 |
|------|------|----------|----------|----------|----------|
| **熊市初期** | 高位放量下跌，均线开始死叉 | 强空 | 2倍 | 5倍 | 30% |
| **熊市中期** | 持续阴跌，反弹乏力 | 偏空 | 3倍 | 5倍 | 50% |
| **熊市后期** | 缩量横盘，恐慌情绪减弱 | 中性偏空 | 3倍 | 3倍 | 30% |

#### 熊市初期
- 禁止逆势做多
- 反弹至前高或EMA压力位即做空
- 止盈目标：前低或关键支撑

#### 熊市中期
- 可轻仓抢反弹，严格止损
- 做空为主，顺势加仓
- 关注成交量萎缩信号

#### 熊市后期
- 降低做空频率，防止底部扫损
- 观察底背离信号
- 为牛市布局做准备

---

### 牛市阶段

| 阶段 | 特征 | 策略偏向 | 做多杠杆 | 做空杠杆 | 仓位上限 |
|------|------|----------|----------|----------|----------|
| **牛市初期** | 底部放量突破，均线开始金叉 | 中性偏多 | 3倍 | 3倍 | 30% |
| **牛市中期** | 趋势确立，逐步走高 | 偏多 | 5倍 | 3倍 | 70% |
| **牛市后期** | 加速上涨，情绪狂热 | 中性偏多 | 3倍 | 3倍 | 50% |

#### 牛市初期
- 轻仓试探，确认趋势
- 回调至EMA支撑位做多
- 严格止损，防止假突破

#### 牛市中期
- 顺势做多，回调加仓
- 减少做空频率
- 止盈目标：前高或扩展位

#### 牛市后期
- 降低做多杠杆，分批止盈
- 警惕顶背离信号
- 禁止追高，设置移动止盈

## 一、空头排列策略

**判断条件**: `ema2 < ema3 < ema4` (EMA144 < EMA169 < EMA576)

### 1. 杠杆配置
- **做空**: 5倍杠杆
- **做多**: 3倍杠杆

### 2. 逆势做多规则
- 2.1 在上一个下跌前的震荡区域 30% 位置(非假突破的时候)，移动止损至开仓价
- 2.2 高点附近完全止盈（资金博弈论：庄家需触发做空止损）

### 3. EMA 距离过滤
- **条件**: `ema2 < ema3` 且 ema2 与 ema4 距离过远
- **规则**: 收盘价 > ema3 → **不做多**（假信号）
- **备选**: 在 ema4 下方 2% 位置挂空单

### 4. 布林带信号过滤
- **条件**: 触碰布林带 + K线信号 + 收盘价跌破近6根K线平均成交价
- **规则**: 忽略该信号
- **案例**: 2025-12-14 16:00:00

### 5. 吞没形态止损
- **条件**: 阴线包阳（看跌吞没）
- **规则**: 止损设置为该K线开盘价

### 6. 成交量过滤
- **条件**: 出现交易信号，但近3根K线成交量递减 `Vol(n-2) > Vol(n-1) > Vol(n)`
- **规则**: 忽略做多信号

### 7. 大振幅K线挂单
- **条件**: 交易信号（非假突破）+ K线振幅 ≥ 3%
- **规则**: 在K线高点到收盘价距离的 **0.382** 位置挂单
- **案例**: 2025-12-05 00:00:00

### 8. 假突破直接做空
- **条件**: 交易信号为假突破信号
- **规则**: 直接市价做空
- **案例**: 2025-12-11 04:00:00

## 三、短线博弈论与消息面策略

### 1. 重大事件挂单策略

**适用场景**: 美联储讲话、CPI数据、非农数据等高波动事件

#### 事件前挂单
| 挂单位置 | 方向 | 距离 | 说明 |
|----------|------|------|------|
| 高点上方 | 做多 | +3% | 突破追多 |
| 低点下方 | 做空 | -3% | 突破追空 |

#### 挂单成交后处理
- **多单成交** → 立即取消空单挂单
- **空单成交** → 立即取消多单挂单
- **收盘确认**: 
  - 多单持仓 + 当根K线收阴 → **平仓**
  - 空单持仓 + 当根K线收阳 → **平仓**

---

### 2. 消息面博弈论

#### 利空消息后的反弹陷阱
- **现象**: 利空消息发布后，价格短暂下跌后快速反弹
- **原因**: 庄家需要扫掉消息面跟风做空的高杠杆止损
- **策略**: 
  - 等待反弹结束后再做空
  - 不在消息公布瞬间追空

#### 利多消息后的回调陷阱
- **现象**: 利多消息发布后，价格短暂上涨后快速回调
- **原因**: 庄家需要扫掉消息面跟风做多的高杠杆止损
- **策略**: 
  - 等待回调结束后再做多
  - 不在消息公布瞬间追多

---

### 3. 消息面后的趋势判断

| 消息面后续 | 预期走势 | 策略 |
|------------|----------|------|
| 一周内无重大消息 | 单边趋势 | 顺势持仓，扩大止盈 |
| 一月内无重大消息 | 强单边趋势 | 加仓顺势，减少逆势 |
| 近期有多个消息事件 | 震荡为主 | 降低杠杆，缩短持仓 |

#### 消息面后单边趋势规则
- **条件**: 重大利空/利多后，1周内无其他重大事件
- **规则**: 
  - 利空后无消息 → 做空持仓周期延长，止盈目标扩大
  - 利多后无消息 → 做多持仓周期延长，止盈目标扩大
- **原理**: 缺少新的消息面催化，趋势更容易延续

## 五、资金管理体系（核心）

### 1. 单笔风险敞口

| 账户状态 | 单笔风险上限 | 说明 |
|----------|--------------|------|
| 正常 | 2% | 单笔最大亏损不超过账户的2% |
| 连亏2次后 | 1% | 降低风险敞口 |
| 连亏4次后 | 0.5% | 进入保护模式 |
| 连亏6次后 | 暂停交易 | 强制休息24小时，复盘后恢复 |

> **铁律**：无论信号多强，单笔风险永不超过2%

### 2. 仓位计算公式

```
仓位金额 = (账户净值 × 风险比例) / |入场价 - 止损价| × 入场价

实际仓位 = min(仓位金额, 账户净值 × 仓位上限)
```

**计算示例**：
- 账户净值：10,000 USDT
- 风险比例：2%
- 入场价：100,000
- 止损价：98,000（止损距离2%）
- 仓位上限：50%

```
理论仓位 = (10000 × 0.02) / 0.02 × 100000 = 10,000 USDT
实际仓位 = min(10000, 10000 × 0.5) = 5,000 USDT
```

### 3. 最大回撤熔断

| 回撤级别 | 触发条件 | 处理措施 |
|----------|----------|----------|
| 一级预警 | 日亏损 ≥ 3% | 当日仓位减半，杠杆降一档 |
| 二级预警 | 周亏损 ≥ 8% | 本周停止新开仓，只能减仓 |
| 三级熔断 | 月亏损 ≥ 15% | 本月停止交易，全面复盘 |
| 绝对熔断 | 总回撤 ≥ 25% | 清空所有仓位，暂停系统 |

### 4. 盈利管理

| 盈利阶段 | 处理规则 |
|----------|----------|
| 账户盈利 +20% | 提取10%利润至冷钱包 |
| 账户盈利 +50% | 提取本金的50% |
| 账户盈利 +100% | 提取全部本金，用利润继续交易 |

## 六、止损止盈量化参数

### 1. 初始止损设置

| 止损类型 | 计算方式 | 适用场景 |
|----------|----------|----------|
| ATR止损 | 入场价 ± 1.5×ATR(14) | 趋势行情 |
| 结构止损 | 前一波段高/低点 ± 0.3% | 关键位置入场 |
| 固定比例 | 入场价 ± 2% | 消息面博弈 |
| 吞没止损 | 吞没K线开盘价 | 形态入场 |

**优先级**：结构止损 > ATR止损 > 固定比例

### 2. 移动止损规则

| 盈利阶段 | 止损调整 |
|----------|----------|
| 盈利 ≥ 1R | 止损移至保本位（入场价 ± 手续费） |
| 盈利 ≥ 1.5R | 止损移至 +0.5R 位置 |
| 盈利 ≥ 2R | 使用 ATR(14) × 1.0 跟踪止损 |
| 盈利 ≥ 3R | 使用 ATR(14) × 0.8 跟踪止损（收紧） |

> R = 初始止损距离（风险单位）

### 3. 止盈目标

| 市场状态 | 目标1 | 目标2 | 目标3 |
|----------|-------|-------|-------|
| 震荡市 | 1.5R | 2R | - |
| 趋势初期 | 2R | 3R | 跟踪 |
| 趋势中期 | 2R | 4R | 跟踪 |
| 消息面 | 1R | 1.5R | - |

### 4. 分批止盈规则

| 到达目标 | 平仓比例 | 止损调整 |
|----------|----------|----------|
| 目标1 | 40% | 移至保本 |
| 目标2 | 30% | 移至目标1位置 |
| 目标3 | 剩余全部 | ATR跟踪 |

### 5. 时间止损

| 持仓时间 | 盈亏状态 | 处理 |
|----------|----------|------|
| 12根K线 | 浮亏 | 减仓50% |
| 24根K线 | 盈亏平衡附近 | 平仓离场 |
| 48根K线 | 未达目标1 | 平仓，视为信号失效 |

## 七、关键参数量化定义

### 1. EMA距离量化

| 参数 | 量化定义 |
|------|----------|
| EMA2与EMA4距离过远 | `abs(ema2 - ema4) / ema4 > 5%` |
| EMA2与EMA4距离适中 | `2% < abs(ema2 - ema4) / ema4 ≤ 5%` |
| EMA2与EMA4距离过近 | `abs(ema2 - ema4) / ema4 ≤ 2%`（震荡市） |

### 2. 震荡区域定义

```
震荡区域识别条件：
1. 近20根K线最高价与最低价的振幅 < 8%
2. ADX(14) < 25
3. EMA2、EMA3、EMA4 三线缠绕（两两距离 < 1%）

震荡区域高点 = 近20根K线的最高价
震荡区域低点 = 近20根K线的最低价
震荡区域30%位置（做多）= 低点 + (高点 - 低点) × 0.3
震荡区域30%位置（做空）= 高点 - (高点 - 低点) × 0.3
```

### 3. 假突破量化定义

```
看涨假突破（用于做空）：
1. K线最高价突破前高（前20根K线高点）
2. 收盘价回落至前高下方
3. 上影线长度 ≥ 实体长度 × 1.5
4. 成交量 > 前3根K线平均成交量 × 1.2

看跌假突破（用于做多）：
1. K线最低价跌破前低（前20根K线低点）
2. 收盘价回升至前低上方
3. 下影线长度 ≥ 实体长度 × 1.5
4. 成交量 > 前3根K线平均成交量 × 1.2
```

### 4. 背离量化定义

```
底背离（看涨）：
1. 价格创新低（近50根K线）
2. RSI(14) 未创新低，形成更高的低点
3. MACD柱状图缩短或金叉
4. 确认条件：下一根K线收阳

顶背离（看空）：
1. 价格创新高（近50根K线）
2. RSI(14) 未创新高，形成更低的高点
3. MACD柱状图缩短或死叉
4. 确认条件：下一根K线收阴
```

### 5. 反弹/回调乏力量化

```
反弹乏力（熊市做空信号）：
1. 反弹幅度 < 前一波下跌幅度 × 0.382
2. 反弹成交量 < 下跌成交量平均值 × 0.7
3. RSI(14) 反弹未突破50
4. 反弹受阻于EMA2/EMA3

回调乏力（牛市做多信号）：
1. 回调幅度 < 前一波上涨幅度 × 0.382
2. 回调成交量 < 上涨成交量平均值 × 0.7
3. RSI(14) 回调未跌破50
4. 回调获支撑于EMA2/EMA3
```

## 八、多周期共振体系

### 1. 周期定义

| 周期层级 | 时间框架 | 作用 |
|----------|----------|------|
| 大周期 | 日线(1D) | 定方向，定趋势 |
| 中周期 | 4小时(4H) | 定入场区域 |
| 小周期 | 1小时(1H) | 精确入场点 |

### 2. 共振规则

| 日线 | 4H | 1H | 操作 | 仓位系数 |
|------|----|----|------|----------|
| 多头排列 | 多头排列 | 多头信号 | 做多 | 100% |
| 多头排列 | 空头排列 | 多头信号 | 做多 | 50% |
| 多头排列 | 空头排列 | 空头信号 | 不操作 | - |
| 空头排列 | 空头排列 | 空头信号 | 做空 | 100% |
| 空头排列 | 多头排列 | 空头信号 | 做空 | 50% |
| 空头排列 | 多头排列 | 多头信号 | 不操作 | - |

### 3. 周期冲突处理

```
规则：大周期优先，逆大周期方向只做快进快出

日线与4H冲突时：
- 跟随日线方向：正常仓位
- 逆日线方向：仓位减半 + 止盈目标减半 + 不加仓

4H与1H冲突时：
- 等待1H与4H同向后入场
- 或在4H关键支撑/阻力位入场，1H确认
```

## 九、震荡市策略

### 1. 震荡市识别

```
满足以下条件判定为震荡市：
1. ADX(14) < 25
2. 布林带带宽(20,2) < 4%
3. EMA2、EMA3、EMA4 距离 < 1%
4. 近20根K线在同一区间内波动
```

### 2. 震荡市参数调整

| 参数 | 趋势市 | 震荡市 |
|------|--------|--------|
| 杠杆 | 3-5倍 | 2-3倍 |
| 仓位 | 50-70% | 30%以下 |
| 止盈目标 | 2-4R | 1-1.5R |
| 持仓周期 | 长 | 短（≤12根K线） |

### 3. 震荡市操作规则

| 位置 | 操作 | 止损 | 止盈 |
|------|------|------|------|
| 区间下沿 + 看涨信号 | 做多 | 区间下沿 - 0.5% | 区间中轨或上沿 |
| 区间上沿 + 看跌信号 | 做空 | 区间上沿 + 0.5% | 区间中轨或下沿 |
| 区间中部 | 不操作 | - | - |

### 4. 震荡转趋势识别

```
向上突破确认：
1. 收盘价 > 震荡区间上沿
```

## 十、复盘体系

### 1. 交易日志记录

每笔交易必须记录：

| 字段 | 说明 |
|------|------|
| 交易时间 | 开仓/平仓时间 |
| 品种 | BTC/ETH等 |
| 方向 | 多/空 |
| 入场价 | 实际成交价 |
| 止损价 | 初始止损 |
| 止盈目标 | 目标1/2/3 |
| 仓位 | 实际仓位金额 |
| 杠杆 | 使用杠杆倍数 |
| 入场理由 | 触发的规则编号 |
| 市场状态 | 多头/空头/震荡 |
| 多周期共振 | 日线/4H/1H状态 |
| 平仓价 | 实际平仓价 |
| 盈亏 | 金额和R倍数 |
| 平仓理由 | 止损/止盈/时间止损/主动平仓 |
| 执行评分 | 1-5分（是否完美执行规则） |
| 复盘笔记 | 改进点 |

### 2. 复盘周期

| 周期 | 内容 | 时间 |
|------|------|------|
| 日复盘 | 检查当日交易执行质量 | 每日收盘后15分钟 |
| 周复盘 | 统计周胜率/盈亏比/回撤 | 周日 |
| 月复盘 | 分析策略表现，调整参数 | 月末 |
| 季复盘 | 评估整体系统，重大调整 | 季末 |

### 3. 关键绩效指标

| 指标 | 健康值 | 预警值 |
|------|--------|--------|
| 胜率 | ≥ 40% | < 35% |
| 盈亏比 | ≥ 2:1 | < 1.5:1 |
| 最大回撤 | < 15% | > 20% |
| 夏普比率 | > 1.5 | < 1.0 |
| 执行准确率 | ≥ 90% | < 80% |
| 平均持仓时间 | 按计划 | 过长/过短 |

### 4. 策略调整规则

```
触发调整条件：
1. 连续10笔交易胜率 < 30%
2. 周回撤 > 8%
3. 盈亏比连续2周 < 1.5
4. 执行准确率 < 80%

调整流程：
1. 暂停新开仓
2. 导出近30笔交易记录
3. 分析亏损交易的共同特征
4. 识别规则漏洞或参数偏差
5. 小仓位测试调整后的规则（至少20笔）
6. 验证通过后恢复正常交易
```
