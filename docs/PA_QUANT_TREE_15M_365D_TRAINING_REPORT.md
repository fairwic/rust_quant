## 市场一致性

趋势策略四市场平均 R：BTC -0.378、ETH -0.217、SOL -0.205、BCH -0.264。

区间策略四市场平均 R：BTC -0.939、ETH -0.707、SOL -0.630、BCH -0.521。

失败方向跨四市场一致，不是单币种或单季度偶然结果。

## 模型竞赛

- 趋势和区间的 M0-M4 one-standard-error 选择均回到 M0。
- all-reject 模型因少于 30 笔或覆盖率低于 10%被淘汰。
- 没有生成 RuntimeManifest Challenger。

## 成本/R 诊断

- 趋势 `cost_r <= 0.10`：522 笔，基础平均 +0.044R、PF 1.074；两倍成本 -0.024R，失败。
- 区间 `cost_r <= 0.20`：29 笔，基础平均 +0.280R、两倍成本 +0.122R；但少于 30 笔，胜率 51.72%、PF 1.191，仍未达到 60%和1.2门槛。
- 以上只形成下一轮训练假设，不得在当前窗口固化后再把同一窗口称为 OOS。

## 决策

1. 旧81天报告只保留为预检，当前结论以本报告为准。
2. v1 趋势和区间策略不得进入 Shadow/Paper。
3. 1h 四市场全年数据已补齐，应建立独立实验与数据指纹，不与15m结果混合。
4. 资金费率接入前，不生成完整 ResearchEvidenceManifest。
