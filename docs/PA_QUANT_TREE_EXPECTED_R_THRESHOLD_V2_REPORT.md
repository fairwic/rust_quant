## 迭代身份

- 训练协议：`pa-training-v2-expected-r-threshold`。
- 变更范围：仅修改 M2 逻辑回归的 keep 阈值选择，不改变 PA 候选、入场、止损、目标、成本、时间切分、CART、Vegas 或组合回放。

## 结论

1. 固定 `P(win) >= 0.5` 确实造成了 M2 的虚假全拒绝；v2 已修复该逻辑漏洞。
2. 修复后只有 `pa_trend_15m` 选择 M2，但验证均值仍为 -0.034R，置信方向没有转正，不能创建 Shadow/Paper Challenger。
3. 当前失败不再能归因于逻辑回归阈值；主要问题仍在候选结构本身与特征解释力。
4. CART/Forest 继续全拒绝，说明当前单层中位数分裂不足以发现正期望叶节点；在新增深树前应先增加经批准、具备机制解释的特征，避免用复杂度拟合噪声。
5. 密封 OOS 保持关闭。下一轮只能在训练期内预注册新 feature/hypothesis 后迭代。
