## 预注册条件核对

| 条件 | 结果 |
| --- | --- |
| Walk-forward平均R大于0 | 通过，但标准误大于均值 |
| 至少30笔且覆盖率不低于10% | 通过，114笔、12.13% |
| 两倍成本后为正 | 失败，-0.079R |
| 四市场多数为正 | 失败，仅SOL/BCH为正 |
| 共享市场bootstrap下界接近或大于0 | 未证明；当前验证均值下界为负 |
| 共享组合回撤低于15% | 全训练诊断通过，但非独立验证 |
| 胜率大于60% | 失败，45.67% |

## 决策

1. v4新增特征形成弱正向筛选信号，说明趋势质量特征方向值得保留在Feature Registry v2。
2. v4不生成Shadow/Paper Challenger，不打开密封OOS。
3. 不在当前窗口继续修改特征公式、概率阈值或symbol参数。
4. 当前下一棒开盘直接入场仍包含大量缺乏后续确认的弱恢复候选。
5. 下一阶段如研究跟随确认，必须创建新策略能力或major版本，保留当前候选与报告，不原地覆盖。
